Project Summary

This project is a memory matching game and its purpose it to test the strength of peoples' memory. With eight different pairs of graphics the user must locate and match the correct corresponding cards. The game has a timer that counts the amount of time the user is using the application as well as a "moves" counter that tracks the numbers of card matching moves they make. This is a simple game that is helpful to strengthen memory as well as test hearing which is something that fades with age. It helps keep your brain sharp and be more aware of sensing sounds. I've provided more details in the "Project Proposal," which is located below.

For my three extras I have chosen to focus on CSS animations for the card flips, a timer, and while not in the first prototype, cookies. As you will see when playing the game, when you click a card it flips. If two cards don't match they turn red and flip back over. But when two cards match they turn over, pulse, and stay visible. There is also a timer that is placed above the board game that tracks the amount of time a player is using the application. It includes both minutes and seconds. I intend to include the third extra in the final which will be cookies. This will be implemented through having the user enter their name before being able to play.
